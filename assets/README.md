# Avatares de usuario

Guardá acá las fotos de perfil con estos nombres EXACTOS:

- `bian.jpg`  → la foto de Bian (la de fondo rosa con anteojos)
- `luke.jpg`  → la foto de Luke (cuando la tengas)

La app las busca en `assets/bian.jpg` y `assets/luke.jpg`.
Si un archivo no existe todavía, se muestra la inicial (B / L) sobre el color del usuario como respaldo —
no se rompe nada.

Recomendado: imágenes cuadradas (1:1), buena calidad (≥ 512×512). Se recortan en círculo automáticamente.
